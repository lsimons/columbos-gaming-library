Dark Sun Campaign Rules By Leo Simons
This file contains my rule modifications to the PHB, MM, DMG, AOH, PO:SP and PO:CT. (that is: new ability score generation methods, the classes adapted to Skills and Powers, new kits, new rules, and more.) This file is completely viewable by both players and DMs, because I wrote it for my own roleplaying group.

Players Option: Psionics and Living Items By Leo Simons
This file contains my changes to the rules for psionic powers, psionic research, psionic items, and, last but not least, new psionic powers.
I'm still working on expanded versions of the rules, like new proficiencies, better combat rules (like the special effects.) However, you might want to take a look at what's already there.
ALso, I'm now done with the living items section, which contains my philosophy regarding living items, rules for their creation, the nature-master character class, and some living items.
If you have any suggestions for modifications of the rules, or a collection of living items, please send them to me.
